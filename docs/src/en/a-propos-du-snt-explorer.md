# About SNT Explorer

SNT Explorer is a dedicated, multi-user decision-support application that helps malaria control programs translate analytical results into concrete, costed subnational plans. It provides a collaborative environment where program teams and technical working groups can explore data and progressively refine intervention plans in a transparent, structured way.

[Discover the full SNT toolbox](https://www.snt-toolbox.org/)

With financial support from the [Gates Foundation](https://www.gatesfoundation.org/), the SNT project is a partnership bringing together [Bluesquare](https://www.bluesquarehub.com/), [PATH](https://www.path.org/), [CHAI](https://www.clintonhealthaccess.org/), [AHADI](https://www.appliedhealthanalytics.org/), [IDM](https://www.idmod.org/), [SwissTPH](https://www.swisstph.ch/en/) and is technically supported by the [WHO](https://www.who.int/).
